# 成就图标需求单

日期：2026-06-29

## 统一生成要求

用途：Unity 成就面板和成就弹窗图标。当前面板中图标显示约为 188x188，建议先生成 1024x1024，再导入 Unity 缩放使用。

统一风格：

- 正方形图标，透明背景 PNG。
- 不要文字、不要数字、不要水印、不要复杂背景。
- 扁平插画或轻厚涂风格，轮廓清楚，形状在 188x188 下仍能看懂。
- 色彩明快但不过度霓虹，适合大学生活、卡牌叙事、轻松讽刺的成就系统。
- 每张图保留一个明确主体，最多加 1 到 2 个辅助元素。
- 未解锁状态会在游戏内做半透明处理，图标本身按“已解锁”状态生成。

可复制给 ChatGPT 的统一前缀：

```text
请生成一个游戏成就图标，1024x1024，透明背景 PNG，扁平插画风格，清晰粗轮廓，温暖但高对比的配色，适合大学生活题材的卡牌游戏 UI。图标中不要出现文字、数字、水印或复杂背景。主体需要在缩小到 188x188 时仍然清楚。
```

## 导入命名建议

建议导入到：

```text
Assets/UniversitySimulator/Art/AchievementIcons/
```

文件名建议和成就 ID 一致，全部小写或全部大写都可以，但要保持统一：

```text
ACH_END_ECON_HIGH.png
ACH_END_ECON_LOW.png
...
ACH_CHOICE_E041_R.png
```

导入 Unity 后建议设置：

| 设置 | 建议值 |
| --- | --- |
| Texture Type | Sprite (2D and UI) |
| Sprite Mode | Single |
| Alpha Is Transparency | 开启 |
| Max Size | 1024 |
| Compression | Normal Quality 或 None |
